### 跨分支复制
`git checkout <branchname> <path>`

### 查看关联分支
- `git branch -vv`
- `cat .\.git\config`

### Clone a Specific Branch
- `git clone -b <branchname> <remote-repo-url>`
- `git clone -b <branchname> --single-branch <remote-repo-url>`
